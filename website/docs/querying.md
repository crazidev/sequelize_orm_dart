# Querying

This guide covers all aspects of querying data with Sequelize Dart, including finding, creating, updating, and deleting records.

## Overview

Sequelize Dart provides a type-safe query builder that makes it easy to interact with your database. All queries use column extensions for full type safety and autocomplete support.
